# gallery-web
The main web app of the gallery app

## Usage
```
$ ./gallery-web serve --help
Starts the serve

Usage:
  gallery-web serve [flags]

Flags:
  -h, --help   help for serve

Global Flags:
      --verbose   Verbose display
```
